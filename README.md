# Student Tools Hub

Free online calculators for students - GPA, CGPA, Percentage, Age, Marks, Grade, and Attendance calculators with modern UI.

## Features

- **7 Calculators**: Percentage, GPA, Age, CGPA, Marks, Grade, Attendance
- **Modern UI**: Clean design with dark mode support
- **Responsive**: Works on all devices
- **Local Storage**: Save calculation history
- **SEO Optimized**: Fast and search engine friendly

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Calculators

1. **Percentage Calculator** - Calculate percentages and percentage changes
2. **GPA Calculator** - Calculate your Grade Point Average
3. **Age Calculator** - Calculate exact age with birthday countdown
4. **CGPA Calculator** - Cumulative GPA across semesters
5. **Marks Calculator** - Total marks, average, and percentage
6. **Grade Calculator** - Convert marks to grades
7. **Attendance Calculator** - Track attendance requirements

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
